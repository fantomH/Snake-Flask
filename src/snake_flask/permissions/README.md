# SnakePermissions

SnakePermissions is a Flask extension providing basic permissions functionalities.

## API

### Default configuration

| KEYS | DEFAULT VALUES | DESCRIPTION |
| :--- | :------------- | :---------- |
| `SNAKE_PERMISSIONS_DATABASE` | `/instance/"permissions.sqlite` | |
| `SNAKE_PERMISSIONS_URL_PREFIX` | `/permissions` | |
| `SNAKE_PERMISSIONS_BASE_TEMPLATE` | `None` | |
